# PROYEK UAS DASAR PEMROGRAMAN

## Fitur yang akan ada apa saja

### 1. Login Akun

Fitur ini membuat user agar bisa mempunyai akun untuk mengakses dengan fitur keamanan yang cukup tinggi, terdapat password juga.

### 2. Multi Wallet

Konsepnya pembagian uang lebih spesifik, misal user ingin membagi menjadi tiga bagian, 50% untuk dana utama, 30% dana darurat, dan sisanya 20% untuk dana
tabungan.

Jadi, ketika ada pengeluaran dari user, dan user sudah mengaktifkan fitur multi wallet, maka akan tampil informasi berupa pengeluaran yang di maksud ini dari mana? apakah dari utama, darurat atau tabungan.

### 3. Spending/Income Report

Ketika user melakukan pengeluaran maupun pemasukan, setelahnya ada informasi lebih lanjut dari pemilihan tersebut, informasi yang disediakan seperti kapan pemasukan/pengeluaran dilakukan tanggal bulan kapan , rincian dana multi wallet yang masih ada sisanya.

### 4. Credit Tracker

Fitur ini dibuat untuk pengguna agar pengguna bisa menambahkan catatan tambahan terkait pelunasan/hutang yang belum diselesaikan, sehingga jika ini diaktifkan maka akan keluar setiap fitur pemasukan / pengeluaran dilakukan, juga ada progress credit tracker ini sebesar apa jika user sudah memasukkan beberapa dana agar bisa lunas sepenuhnya.

### 5. Savings Tracker

Sama seperti fitur Credit Tracker, fitur ini dibuat untuk menyediakan informasi terkait dana tabungan untuk apa, misal untuk tiket konser maka akan ditampilkan progress savings yang sudah dilakukan.

### 6. Budgeting

Budgeting adalah fitur untuk pengguna agar bisa membuat target yang diinginkan seberapa besar, jadi dengan adanya fitur ini, pengguna dapat mengingat kembali target pemasukan/pengeluaran yang diinginkan hanya beberapa saja atau tergantung penggunanya.
